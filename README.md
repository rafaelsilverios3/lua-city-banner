# Lua City Banner

Arquivos prontos para publicar no GitHub Pages da conta:

```text
https://github.com/rafaelsilverios3
```

Repositorio recomendado:

```text
lua-city-banner
```

Depois de publicar pelo GitHub Pages, a URL final do banner fica assim:

```text
https://rafaelsilverios3.github.io/lua-city-banner/nuvem/banner.png
```

Use essa URL no `server.cfg`:

```cfg
sets banner_detail "https://rafaelsilverios3.github.io/lua-city-banner/nuvem/banner.png"
sets banner_connecting "https://rafaelsilverios3.github.io/lua-city-banner/nuvem/banner.png"
```

Passos no GitHub:

1. Crie um repositorio publico chamado `lua-city-banner`.
2. Envie os arquivos `index.html`, `README.md` e a pasta `nuvem`.
3. Entre em `Settings > Pages`.
4. Em `Build and deployment`, selecione `Deploy from a branch`.
5. Em `Branch`, selecione `main` e `/root`.
6. Salve e aguarde o GitHub liberar a URL.
